ASSISTENTE PESSOAL VIP — ATUALIZAÇÃO GÊMEO FINANCEIRO 0.2.0

PRÉ-REQUISITO
- O GitHub deve estar com a versão Genesis 0.1.2.
- Faça um backup em GUARDIÃO antes de atualizar.

COMO INSTALAR
1. Extraia este ZIP no computador.
2. No GitHub, abra o repositório do aplicativo.
3. Clique em Add file > Upload files.
4. Envie TODOS os arquivos e pastas deste pacote para a raiz do repositório.
5. Preserve a estrutura das pastas js e js/modules.
6. Confirme a substituição dos arquivos com o mesmo nome.
7. Clique em Commit changes.
8. Após o GitHub Pages atualizar, feche totalmente a PWA e abra novamente.

NOVO ARQUIVO OBRIGATÓRIO
- js/modules/financeiro.js

O QUE FOI ADICIONADO
- Entradas e gastos
- Saldo geral estimado
- Contas e categorias
- Orçamento mensal por categoria
- Compromissos recorrentes
- Projeção mensal
- Metas financeiras
- Integração com AGORA, QG, ORÁCULO e VIP GUIA
- Dados financeiros incluídos no backup do GUARDIÃO

DADOS EXISTENTES
A atualização do IndexedDB preserva os dados atuais. Não apague os arquivos antigos manualmente; apenas substitua os arquivos indicados pelo pacote.
